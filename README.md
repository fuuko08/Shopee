# Shopee

## Description

An E-commerce shopping application built with C# language, and Asp.Net Core 7 framework. App run on SQL server and follows MVC pattern.

## Usage

* C#
* Asp.Net Core 7
* Razor
* Stripe
* SQL server
* MVC pattern
